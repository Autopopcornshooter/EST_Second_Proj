## 📌 기능 명세서

### <기능 명세서>
1. 메인페이지
    - 간단한 서비스 소개
    - 로그인 버튼

2. 회원 관리
    - 회원 가입
        - 일반 회원가입
        - SNS 로그인 (구글, 기타 SNS)
        - 아이디/비밀번호 찾기 (옵션)

    - 로그인
        - 기존 회원 로그인
        - SNS 로그인 연동

4. 동네 인증
    - 사용자의 위치 기반 동네 인증

5. 동네 소개
    - 식당 리스트 조회
    - 식당 등록
    - 검색 기능
    - 채팅 기능 접근

6. 식당 등록
    - 식당 또는 음식 이미지 업로드
    - 대표 메뉴 입력
    - 가격 입력
    - 대표 메뉴 설명 입력
    - 가게 위치 등록
      - 가게 위치는 동네 인증 한 곳으로 설정
    - 채팅 기능 연동

7. 채팅
    - 채팅 로그 표시
    - 채팅 메시지 전송
    - 채팅 메시지 삭제
    - 사이드바에 채팅 목록 표시

- 소셜 로그인

    - 구글 map API를 사용합니다.
